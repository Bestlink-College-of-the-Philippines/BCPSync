# BCPSync

   A simple app that sync data from Princetech Integrated School Management System Database to BCP Backend Servers, 
   Built with Node.js

## MODULES
    
   - [x] Grade Sync
   - [ ] Outstanding Balance Sync
   - [ ] SOA 
   - [x] Update BCP Backend Server
